Different box-sizings

The two blue boxes below use different box-sizing values. Both are set to a width of 60px.

Thing 1 is border-box. Its width and height control the final size. The thing 1 box's padding(5 on each side) and border(1 on each side) are subtracted from the 60 pixels it occupies. So the blue content area is 48pixels wide.

Thing 1

Thing 2 is content-box. Its width and height control the size of just the content. The thing 1 box's padding(5 on each side) and border(1 on each side) are added from the 60 pixels it wants. So the total space it occupies is 72px.

Thing 2
